## Структура проекта

```
├── generator/ # генератор CSV-файла
│   ├── generate.py
│   └── Dockerfile
├── report_creator/ # генератор HTML-отчёта
│   ├── report.js
│   ├── package.json
│   └── Dockerfile
├── data/ # папка для CSV и HTML (создаётся автоматически)
├── local_data/  # для локальной откладки генератора данных
├── run.sh # основной скрипт
└── README.md 
```

## Основные команды проекта

Все действия выполняются из корня проекта. Для того чтобы увидеть список всех доступных команд необходимо запустить `./run.sh` без аргументов.

### Сборка образов
```bash
./run.sh build_generator # собирает Docker-образ генератора
./run.sh build_reporter  # собирает Docker-образ аналитика
```

### Генерация данных и отчёта
```bash
./run.sh run_generator  # запускает контейнер, создаёт data/data.csv
./run.sh run_reporter   # запускает контейнер, создаёт data/report.html
```
### Проверка видимости данных из контейнеров
```bash
./run.sh inside_generator   # показывает содержимое /data внутри контейнера генератора
./run.sh inside_reporter    # аналогично предыдущей команде, но для контейнера аналитика
```
### Локальная откладка
```bash
./run.sh create_local_data  # cоздаётся local_data/data.csv
```

### Просмотр структуры
```bash
./run.sh structure  # показывает структуру всех файлов и директорий, начиная с текущей папки 
```

### Просмотр структуры
```bash
./run.sh clear_data # удаляет все файлы типа .csv и .html из data/
```

### Веб-сервис с отчётом
```bash
./run.sh report_server  # запускает контейнер nginx с веб-сервером, раздающий report.html
```
